# 第6回
## もぐらたたき
### ゲーム概要
- 動くモグラの画像を一定回数(10回以上)たたくとゲームクリア
- クリアするとモグラの移動速度が速くなっていく

### 操作方法
- マウスの左クリックで操作

###　クリア条件
- モグラを10回以上クリックすること
- エンドレスかつ難易度上昇によりクリア状況は人それぞれ
